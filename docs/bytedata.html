class riemann.tx.ByteData¶

Wrapper class for byte-like data Iterable and subscriptable (by iterating and subscribing to wrapped data) Can be made immutable self._bytes is a bytearray object when mutable self._bytes is a byte object when immutable Should be mostly transparent to the user Can be treated like bytes or a bytearray in most cases

static validate_bytes(data: Any, length: Optional[int] = 4)¶

Raises ValueError if data is not bytes. Raises ValueError if len(data) is not length. Length may be None for unknown lengths (e.g. scripts). length=None will allow 0 length data.
